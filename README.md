# EcoElectricity

> 🔗 [github.com/Jasurios/EcoElectricity](https://github.com/Jasurios/EcoElectricity)

Интерактивная 3D-карта электростанций мира на спутниковых снимках с рельефом местности, зонами загрязнения и подробной информацией по каждой станции.

## Возможности

- **3D спутниковая карта** на тайлах Esri World Imagery с объёмным рельефом (AWS Terrarium DEM)
- **38+ электростанций** по всему миру: ГЭС, АЭС, угольные, газовые, солнечные и ветровые станции
- **Маркеры с уникальными иконками** под каждый тип станции (капля, атом, дымоход, пламя, солнце, турбина)
- **Зоны воздействия на карте**:
  - зоны загрязнения вокруг угольных и газовых станций
  - зона отчуждения вокруг АЭС
  - зона шумового загрязнения вокруг ветропарков
- **Боковая панель** с фото, описанием, мощностью, годовой выработкой, страной, рекой и количеством агрегатов
- **Переключение языка** EN / RU для всего интерфейса

## Стек

- React 18 + Vite
- MapLibre GL JS (рендер карты, 3D-рельеф, кастомные слои)
- CSS Modules

## Запуск

```bash
git clone https://github.com/Jasurios/EcoElectricity.git
cd EcoElectricity
npm install
npm run dev
```

Откройте адрес, который выведет Vite (по умолчанию `http://localhost:5173`).

## Сборка

```bash
npm run build    # production-сборка в dist/
npm run preview  # просмотр собранной версии локально
```

## Структура проекта

```
src/

```
